Project Overview

ENELOOP is a service that provides rentable and rechargeable batteries for smart devices charging. The service depends on WeChat whiich can provides Secure identity authentication. The service is aimed to help people who meet an emergency or is not convenient to carry a power bank. The service provides users a map to help them find power bank site nearby. This service provides a wallet system. Rentals without a deposit are achieved by requiring a wallet balance. The service provides exception feedback for something go wrong.
